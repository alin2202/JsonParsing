JsonParsing

This program is created for parsing a specific type of .txt file to Python Json object. 
Program also extracts specified data from Json and prints it to the console.

To run this project you'll need:

a) Python installed on your machine (https://www.python.org/downloads/)

b) Extract content of this folder to your local machine 
or clone it from GitHub: git clone https://github.com/alin2202/JsonParsing.git

c) Navigate to newly created folder and run Program.py passing  in the path/fiename of source file to parse:

on Mac OS: python Program.py test_file.txt
on WIndows: Program.py test_file.txt

d) You should see program output in console
